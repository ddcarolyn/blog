⑤ 审计方法论 + 工具栈（你的常驻 checklist）

五维度系统化框架，工具刻意压在免费/高性价比档，覆盖 Keychron 这种中大型站 90% 需求。

免费工具栈（先接这些）
  • Google Search Console — 索引覆盖、Core Web Vitals 真实数据(CrUX)、富媒体增强报告
  • Lighthouse / PageSpeed Insights — 实验室性能基线
  • Screaming Frog（免费 500 URL） — 断链/重复 title/缺 meta/孤儿页
  • Microsoft Clarity — 全站热图/滚动图/session 录制
  • Ahrefs Webmaster Tools（免费） — 月度健康扫描兜底
  • Rich Results Test + validator.schema.org — 结构化数据校验
第一周 → 第一月 节奏
  • 第一周（诊断）：接 GSC + 提交 4 个 sitemap；埋 Clarity；PSI 跑首页+1 collection+3 PDP 记录移动端基线；Rich Results 抽查 10 个 PDP（重复 schema / 缺 aggregateRating）
  • 第二周（量化）：Clarity 看 PDP 变体选择器与 cart→checkout 死区；review mining 提购买障碍；GA4 漏斗对照行业 benchmark
  • 第三–四周（排期）：Screaming Frog 全量爬出孤儿/重复/断链；产出 Impact×Ease 加权 backlog；先修结构性漏斗与索引膨胀

行业对照基线：Shopify 平均 CVR 2.99%（移动 2.87% < 桌面 4.51%）、加购 7.23%、结账放弃 49.8%。Shopify 慢站第一元凶是 app 过多（平均拖慢 40–60%）——先查 app 体积。

数据治理彩蛋：/products.json 公开可抓（变体/库存/价格全暴露）。正向用：写定时脚本拉它，自动追踪 222 个售罄 SKU 的进出 + 竞品价格，变成运营日报数据源（本项目的产品数据库就是这么来的）。
