Keychron SKU 命名解码手册

从内部条码表的 8,730 个真实 SKU 反推出来的命名规律——让你看一眼 SKU（如 K2M-A3-JIS）就知道它是什么产品、什么轴、什么布局。

数据来源：《Keychron 所有机型的条码.xlsx》278 个 sheet / 8,730 条 SKU 实测归纳 ｜ 配套文件：SKU命名解码字典.xlsx（字典 + 全量 8730 行可筛选解析 + K系列尺寸表 + 型号→官网链接）

0. 先看一眼：SKU 长这样

K2M - A3 - JIS

K2 + M
型号：系列(K) + 编号(2) + 尾缀(M=Max)。
→ Keychron K2 Max
A3
变体：字母(A=某配色/背光族，每个型号自己定) + 数字(3=茶轴)。
JIS
地区后缀(可选)：JIS=日本布局。没有后缀=默认美国 ANSI。

一句话：SKU = 型号（系列+编号+尾缀） — 变体字母（颜色/版本，型号内部自定） + 数字（轴体）[-地区]。其中最该记的是数字=轴体和尾缀=配置档，变体字母不用背、看中文信息列即可。

1. 三步读懂任意一个 SKU

第 1 步 · 看破折号前
拆成「系列字母 + 数字 + 尾缀字母」。
系列告诉你哪条产品线（K/Q/V/B/C/M…），尾缀告诉你什么档（Pro/Max/Ultra/HE/基础）。
第 2 步 · 看第一个破折号后的「字母+数字」
数字 = 轴体（1红 2青 3茶 4香蕉 6静音红…，最稳）。字母是该型号自定的颜色/版本族，记不住就对照中文信息列。
第 3 步 · 看第二个破折号后（若有）
那是地区布局后缀（JIS 日本 / UK 英国 / DE 德国…）。没有 = 默认美国 ANSI。-SC = 试产样品，不是地区。

2. 代号对照表（全部从真实数据归纳）

① 系列字母（SKU 最前面）

字母含义SKU 占比
K主力无线键盘（走量、性价比、Mac 友好）最多(~38%)
Q全金属 gasket 客制旗舰（QMK，最贵）~24%
V平价客制（Q 的塑料壳版本）~13%
B超薄预算入门~6%
C有线入门（最便宜）~5%
M鼠标（不是键盘！）~3%
LLemokey（游戏子品牌）~1%
J / G / R / T / F / BM…子品牌(JamesDonkey 等)、人体工学、联名或细分线少量

② 型号尾缀（编号后面的字母，如 K2M）——决定配置档

尾缀含义SKU 数
（无）基础款（初代/标准，常为有线或蓝牙双模）2916
MMax：三模无线 + QMK/VIA + 热插拔，当前主力配置1878
PPro：在基础款上加 QMK/VIA + 热插拔1532
UUltra：顶配（常带小屏/更高规格）844
HHE：Hall Effect 磁轴，可调触发点，游戏向549
XX：特定精简/渠道版327
SESE / Retro SE：特别版 / 复古配色80

还有少量组合尾缀（PX、HW、UW、MG…）是「档位 + 区域/版本」叠加，遇到时以型号中文名为准。升级阶梯大致是：基础 → Pro（加客制）→ Max（加无线）→ Ultra（顶配）；HE 是平行的磁轴分支。

③ 轴体数字 ★最稳的规律（变体字母后面的数字，整机）

数字轴体手感
1红轴 Red线性顺滑（HE 款里 1 常是"星云/磁轴"主款）
2青轴 Blue段落带响（咔哒声）
3茶轴 Brown段落有反馈
4香蕉轴 Banana线性偏顺
5光轴 / 香蕉—
6静音红 Silent Red线性 + 静音
7 / 9静音茶 / 静音香蕉静音系

为什么是这个顺序（猜测）：大致是 Keychron 轴体上市的先后顺序——最早的 Gateron 红/青/茶就是 1/2/3，香蕉(4) 是后来加的主推线性轴，静音系列(6/7/9) 更晚，所以排到后面。数字越小越"经典/老"。

④ 套件颜色数字（barebone 套件时，同一位置的数字变成"颜色"）

这是最容易踩坑的点：当 SKU 是不带轴的「套件 barebone」时，那个数字不再是轴体，而是机身颜色。

数字套件颜色
1黑色
2灰色
3蓝色
4白色
5/6白 / 绿 / 银（较杂）

判断方法：中文信息里有"套件"且没有任何轴体词 → 数字读作颜色；否则数字读作轴体。颜色 1=黑 是默认色，所以排第一。

⑤ 变体字母（第一个破折号后的字母）——⚠ 不要全局背

这是最"乱"的一段，也是你觉得看不懂的根源：同一个字母在不同型号里含义不同。例如 F 在 Q1 Pro 里是"欧洲 ISO"，在 K2 HE 里是"黑色特别版"，在 V1 Max 里是"白色"。数据印证：字母 M/H/K/D 在全量里都横跨 ANSI/ISO/JIS 多种含义。

结论：变体字母是"型号内部自定义的配色/布局/版本族"，是 model-local 的，没有全局规律。实操上别去背它——要么看中文信息列，要么用配套 Excel 的「全量SKU解析」表筛选。它唯一的弱规律是：团队往往按"区域块"或"上市批次"分配字母（如某型号 B=ANSI、F=ISO、H=JIS）。

⑥ 地区后缀（第二个破折号后，如 -JIS）——全局通用、可背

后缀地区后缀地区后缀地区
JIS/JP日本UK英国DE德国
FR法国ES/SP西班牙IT意大利
KR韩国RU俄罗斯UA乌克兰

没有后缀 = 默认美国 ANSI 布局。这些后缀是键盘语言布局的本地化版本，全局一致，可放心背。

3. 专门回答你的困惑：K0–K19 到底什么区别？

K 后面的数字不是"第几代 / 越大越新越好"，而是"不同尺寸 × 高度"的型号编号。而且早期有个非常漂亮的规律——

★ K 系列奇数 = 低 profile 超薄，偶数 = 常规高度（K1 低/K2 常规/K3 低/K4 常规…一路精确交替）。所以看到奇数 K 就知道它是矮薄款。

型号主力布局高度型号主力布局高度
K180%/TKL低profileK960%低profile
K275%常规·经典款K10100% 全尺寸常规
K375%低profileK1165%低profile
K496%常规K1260%常规
K5100% 全尺寸低profileK1380%低profile
K665%常规K1470%常规
K765%低profileK1575%低profile
K880%/TKL常规K1796%低profile

每个 K 型号再叠加尾缀做"配置档"：比如 K2(基础双模) → K2 Pro(加 QMK+热插拔) → K2 Max(再加三模无线) → K2 HE(磁轴版) → K2 Ultra(顶配)。所以"K2 Max"和"K8 Pro"的区别 = 尺寸不同(75% vs 80%) + 档位不同(Max vs Pro)，两个维度一拆就清楚了。

SKU拆解=什么产品
K2M-A3K2 + M(Max) ｜ A(白光族) ｜ 3(茶轴)K2 Max，75% 常规，白光茶轴，美国 ANSI
Q1P-F1Q1 + P(Pro) ｜ F(欧洲族) ｜ 1(红轴)Q1 Pro 客制旗舰，欧洲 ISO 布局，红轴
K2H-J4K2 + H(HE 磁轴) ｜ J(黑色族) ｜ 4(此处为"快速青柠"磁轴)K2 HE 磁轴版，黑色，快速青柠磁轴（HE 款数字含义偏磁轴专属，以中文信息为准）
V1M-Z6V1 + M(Max) ｜ Z(套件族) ｜ 6(套件→颜色)V1 Max barebone 套件（不含轴/键帽），白色系

Caveats & 说明

  • 数据归纳 全部规律来自对 8,730 个真实 SKU 的频次归纳，不是猜的官方文档；个别老型号/联名款可能不完全守规。
  • 字母 model-local 变体字母没有全局含义，务必结合中文信息列；本手册只给"弱规律"。
  • URL 为弱匹配 「型号→官网链接」是按型号名首次匹配，约 60% 命中，个别可能指到同系列的变体/配件页；要精确请用产品数据库按型号搜。
  • HE/特殊 磁轴(HE)与部分特殊款的数字编码偏专属，以中文信息为准。
