模块3 · 机械键盘解剖 · TASK 3.3

键帽：材质 / 工艺 / profile

键帽不参与"按下/触发"这件事，却决定了你 90% 的指腹触感、键盘是"清脆 clack"还是"低沉 thock"、用半年后会不会打油发亮、以及字符会不会被磨掉。本页只回答一个窄问题：ABS vs PBT、double-shot vs dye-sub、OEM/Cherry/SA 三套坐标，到底各自换来什么、牺牲什么——并把营销话术和真实手感拆开。

调研日期：2026-06 ｜ 数据源优先级：UGC（KeebTalk / Deskthority / r/MechanicalKeyboards 共识）> 第三方评测（MKBGuide / Switch and Click / Kineticlabs）> 官方话术（CORSAIR / GMK 经销商页）
主要来源：KeebTalk 论坛原帖、rgbkeys、MKBGuide、Kineticlabs、Switch and Click、CORSAIR、HHKB 官方博客（详见文末编号索引）

1. 材质轴：ABS vs PBT，争的是"耐久 vs 上色自由"

论述

键帽材质之争表面上是"哪个更好"，本质是两条工程曲线的取舍：PBT（聚对苯二甲酸丁二醇酯）更硬、更密、抗磨抗黄、不易打油，但收缩率高、难做鲜艳大色块；ABS（丙烯腈-丁二烯-苯乙烯）更软、上色自由、能做 double-shot 透光与饱和配色，代价是几个月就开始"打油"（shine，表面被指腹磨出油亮反光）。

需要先打掉一个常见误解："PBT 一定比 ABS 声音好/手感好"是把材质和厚度混为一谈。真正影响声音的第一变量往往是键帽壁厚而不是材质——这点社区评测高度一致。

证据

"PBT 材料密度更高（比 ABS 高约 30%），这导致了不同的声音传播特性……在 profile 和厚度相同的前提下，PBT 版本会比 ABS 吸收更多声波。"

— 用户 Surefoot · KeebTalk 论坛《Does Thick ABS perform the same as Thick PBT?》 · keebtalk.com/t/...20286 · 采集 2026-06-10

"ABS 通常音调更高（higher in pitch）但比 PBT 更闷（more muted）。"

— 用户 ventamora · 同一 KeebTalk 帖 · keebtalk.com/t/...20286 · 采集 2026-06-10

"它（PBT）比 ABS 硬，所以其他条件相同的话本应音调更高——但其他条件通常并不相同。"（点破了"thock 全靠 PBT"的简化叙事）

— 用户 fanf · 同一 KeebTalk 帖 · keebtalk.com/t/...20286 · 采集 2026-06-10

第三方评测把"厚度才是关键"说得更直接：

"键帽厚度是影响声音的关键变量，与材质无关——更厚的键帽通常声音更好、更扎实……预算套件无论 ABS 还是 PBT 常为压成本而偷薄，这往往才是声音让人失望的真正原因，而不是材质本身。"

— MKBGuide《PBT vs ABS Keycaps》 · mkbguide.com/blog/pbt-vs-abs-keycaps · 采集 2026-06-10

具体数据

  • PBT 密度比 ABS 高约 30%（Surefoot / KeebTalk）→ 同厚度同 profile 下 PBT 音调更低、共振更少。
  • ABS 打油时间：数月；PBT 保持磨砂：5 年以上（rgbkeys、makeuseof 与 r/MK 共识）。
  • GMK（行业标杆 ABS 厂）键帽壁厚约 1.5mm——证明高端 ABS 靠"厚 + double-shot"打出"creamy/thocky"声，而非材质天生输。来源：UniKeys / mechanical-keyboard.com。
  • 厚度档差感知阈：1.0mm 与 1.5mm 之间差异明显可闻（MKBGuide）。

小结：材质决定的是"耐久 + 上色路线"，不是简单的"谁声音好"。PBT 抗磨抗油、天生偏低沉，适合"买一套用很久、深色键帽党"；ABS 上色自由、可做透光与饱和配色（GMK 生态），用厚壁 + double-shot 把声音和质感拉到高端，代价是会打油。买之前先看壁厚，再看材质。

2. 工艺轴：double-shot vs dye-sub，争的是"字符永不掉 vs 图案随便印"

论述

这一轴决定字符（legend）怎么上去、能撑多久、能不能透光、能做多复杂的图案，且与材质正交（ABS/PBT 都能 double-shot，PBT 更常 dye-sub）。

  • Double-shot（二色注塑）：两层塑料分别注塑，字符是"实体嵌进键帽本体"，物理上磨不掉、可做 RGB 透光；但每个颜色要一套模具，颜色组合受限、成本高。
  • Dye-sub（热升华染色）：染料受热渗入键帽表层，像高分辨率打印机，能印任意图案/渐变/多色；但染料只能"深处走"，通常只能深底浅字（浅底深字难做，需反向工艺），且染料挡光，基本不支持 RGB 透光。

证据

"Double-shot 在耐久上占上风，因为字符在物理意义上就是键帽的一部分，几乎不可能被磨掉。Dye-sub 虽然非常耐用，但在多年重度使用、尤其接触强清洁剂或过度摩擦后，可能轻微褪色。"

— Kineticlabs《Dye-Sublimation vs Double-Shot Legends》 · kineticlabs.com/blog/...double-shot-legends · 采集 2026-06-10

"Dye-sub 能像高分辨率打印一样印出任意图像、渐变、图案或多色设计；而 double-shot 每个颜色都需要一套独立塑料模具，因此每颗键帽被限制在一到两种纯色。"

— Switch and Click《Double-Shot vs Dye-Sub Keycaps》 · switchandclick.com/double-shot-vs-dye-sub-keycaps · 采集 2026-06-10

"Dye-sub 键帽通常不支持 RGB 灯光，因为染料会挡住光线穿过键帽，最适合无 RGB 的配置。"

— 多家评测共识，原句出自 Switch and Click / Kineticlabs · 采集 2026-06-10

需要识别的营销话术：经销商常把 dye-sub 与 double-shot 都笼统称"耐用、字符不掉"。事实更细——dye-sub 字符会随整个键帽顶面一起缓慢磨损/褪色，只有 double-shot 是"字符永不掉"。

具体数据

  • 字符锐利度：double-shot > dye-sub（dye-sub 受热扩散，边缘略糊）。来源：Switch and Click。
  • 颜色方向：double-shot 任意配色 + 可透光；dye-sub 基本只能深底浅字、不透光。
  • 成本：dye-sub 通常更便宜（无需多套模具），double-shot 偏贵且随键数/布局复杂度上升。来源：Switch and Click / Kineticlabs。
  • 耐久排序：double-shot（永不掉）> dye-sub（极耐用、超长期可能褪）> pad-print/激光（最易掉）。

小结：要 RGB 透光 / 饱和撞色 / 字符绝对不掉 → double-shot（GMK 生态典型）。要 复杂图案、渐变、主题套、预算友好 → dye-sub（多在 PBT 上）。两者都远胜激光刻/丝印这类"几个月掉字"的廉价工艺。

3. profile 轴：OEM / Cherry / SA，争的是"上手即用 vs 极致声音/复古"

论述

profile = 键帽的高度 + 弧面形状 + 是否分行雕刻（sculpted）。它直接改变手指落点、打字角度、误触率和声音音调。三套最有代表性的坐标：

  • OEM：出厂默认最常见（Corsair/Razer/Logitech 等预装都用它），中等高度、分行雕刻、顶面微凹带角度——"标准、零学习成本"，但在客制圈被视为"平庸/出厂感"。
  • Cherry：比 OEM 矮约 2mm，同样分行雕刻。客制圈最爱——矮 = 更少 wobble、打字角度更舒服。MKBGuide 称其在客制市场占约 80% 份额。
  • SA：球面顶（Spherical）、最高、复古打字机外观。声音更低沉饱满、爽感强，但学习曲线陡（1–2 周适应、初期降速），且高度放大噪音、长打可能疲劳——典型的"两极分化"profile。

证据

"Cherry profile 在客制和发烧友市场占主导，约 80% 市场份额……OEM 因与出厂/预装绑定，在发烧友眼里显得平庸（pedestrian）。"

— MKBGuide《Keycap Profiles Explained》 · mkbguide.com/blog/keycap-profiles-explained · 采集 2026-06-10

"SA 引发两极反应，因为它学习曲线陡峭：需要 1–2 周适应，你的打字速度会明显下降。"

— MKBGuide《Keycap Profiles Explained》 · mkbguide.com/blog/keycap-profiles-explained · 采集 2026-06-10

"由于高度原因，SA 键帽可能不如其他选项舒适，也不是为速度优化的……高大的 SA 倾向于放大轴的噪音，产生更响、音调更高的 clack，吸引喜欢复古、强反馈的发烧友。"

— Keychron / apos.audio 综合 profile 指南 · keychron.com/blogs/news/keycap-profiles · 采集 2026-06-10

关于"SA 更响/更高频" vs "更低沉"——评测口径有分歧：球面高 profile 本体共振低沉，但放大了轴和敲击的高频 clack，主观体验取决于整机配置。本页如实并列，不替读者下定论。

5. 关键洞察与产品机会

洞察 1 · "PBT vs ABS 谁声音好"是个伪命题，真变量是壁厚。 多个 KeebTalk 资深用户与 MKBGuide 都指出：预算套件偷薄才是声音差的真因。产品机会：预算/中端键帽把"壁厚 ≥1.4mm"做成显性卖点（像轴标克数一样标厚度），能在一片"PBT/ABS"口水仗里建立差异化信任。

洞察 2 · 工艺的真实痛点是"dye-sub 只能深底浅字 + 不透光"。 想要浅色键帽 + RGB 的用户被结构性卡住，只能贵价 double-shot。产品机会：主推"浅底深字 reverse dye-sub + 半透 PBT 配方"或"PBT double-shot 透光"套件，吃下"既要 PBT 耐久、又要浅色 RGB"这块被忽视的需求。

洞察 3 · profile 是退货/二手流转的高发区——SA 学习曲线陡、OEM 被嫌平庸。 新手常因不知道 profile 会改变手感而买错。产品机会：预装键盘默认上 Cherry profile（客制圈 80% 偏好、矮舒适）即可显著提升开箱好评；同时卖"profile 体验小样卡"（每种 profile 几颗）降低试错成本。

洞察 4 · 教育缺口=内容机会。 营销页普遍把 dye-sub/double-shot 笼统说成"耐用不掉字"，掩盖了 dye-sub 会随顶面缓慢褪、SA 会放大噪音等真相。产品机会：诚实的"三轴对照 + 壁厚透明 + profile 体验"内容/导购，正是 ddcarolyn 这类"给模板不给作业"的差异化切口。

Caveats & 证伪状态

  • 已证伪 "PBT 一定比 ABS 声音更好/手感更好"被推翻：KeebTalk 用户（fanf、Surefoot）与 MKBGuide 一致指出，壁厚才是声音第一变量，材质只在"同厚同 profile"时才比较有意义；GMK 用 1.5mm 厚 ABS double-shot 做出公认高端"creamy/thocky"声即反例。
  • 已证伪 "PBT 永不打油/永不变"被推翻：rgbkeys 明确写"没有塑料完全免疫磨损"，PBT 只是"很慢"，Deskthority 用户观察到极重度使用下空格等键仍会被磨亮。准确说法是"PBT 抗油远强于 ABS、可保磨砂 5 年+"，而非绝对。
  • 已证伪 "dye-sub 和 double-shot 一样字符永不掉"被推翻：Kineticlabs 指出 dye-sub 字符会随整个顶面缓慢磨损/褪色，只有 double-shot 是物理永久。经销商话术常模糊这一点。
  • 部分成立 profile 高度数值跨来源不一致（Cherry 7.5–9.4mm、SA 13–16.5mm），因量取基准不同。本页给区间不给单一"权威值"，图2 为比例示意而非精确测绘。
  • 已核实 Cherry profile 在客制圈占主导（MKBGuide 称约 80%），OEM 为预装默认、SA 学习曲线 1–2 周——多源一致。
